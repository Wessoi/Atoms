Updates

Learn more about the recent updates that have occured on the Atom Survival Server.

  • POSTPONED 1.14 SERVER UPDATE: We are currently postponing the 1.14 update to later this week so people to not lose anything and so we have all our files still in tact without any errors or issues. This would also, allow it so we can use plugins still and everyone will still have their commands because, Spigot 1.14 comes out later around this week.
  • CAPACITY UPDATE & COMMAND UPDATES: We've highered the max capacity of the server from 16 to 25. We have also, changed the difficulty from normal to hard for the more fun experience on the server because we felt that normal was too easy for many players. Along with that we have changed the /rules command a little and have also changed how ranks and the tablist look like. We are currently working on report system yo report cheaters, hackers, griefers, or raiders, or even bad staff. We are also planning on adding a scoreboard back along with adding a death counter and changing the /help layout a lot more and adding new /plugins and /sites commands to the server.
  • REMOVAL OF OFFICAL SHOPS & NEW AREAS: We have decided due to everyone not going to the fact that we are having a official shop. We are removing the official shop and all shops will now be specifically player based on the server. We've also, added a warp and crates area that are currently going under construction right now in the spawn area.
